How it works?

  1. SHA512 100 times the Password
  2. Convert string to UTF8 bytes
  3. Convert the bytes into Base64
  4. Add "I'm nothing like yall" to the beginning of the string
  5. AES encrypt the string
  6. Compress the string with GZIP
  7. DES encrypt the string
  8. Compress again with GZIP
  9. 3DES encrypt the string

That's it.
